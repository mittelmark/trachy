# trachy R package ![](../../actions/workflows/r.yml/badge.svg)


R package for the analysis of Trachypithecus microbiome data for the paper
"Molecular genetic characterization of the diet of limestone and rainforest langurs" (Truong et al. 2026 - in preparation).

The package contains the data and the code for that paper.

## Build and install the package from the terminal

To build, check and install this package you need an installed R and the sbi package. 

The steps after you installed R are as follows:


```
$ R 
> install.packages("https://github.com/mittelmark/sbi/releases/download/v0.4.2/sbi_0.4.2.tar.gz",repos=NULL)
> install.packages("https://github.com/mittelmark/trachy/releases/download/v0.0.4/trachy_0.0.4.tar.gz",repos=NULL)

```
